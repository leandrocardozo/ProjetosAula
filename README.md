## Sobre os Projetos

Este repositório contém projetos desenvolvidos durante o curso da B7Web. Embora não sejam projetos de autoria original, foram fundamentais para meu aprendizado em desenvolvimento web.

Durante a construção destes projetos, digitei cada linha de código, testei, refatorei e busquei entender o funcionamento de cada parte. Embora nem tudo tenha sido absorvido 100% de imediato, grande parte dos conceitos já fazem parte do meu repertório técnico.

Estes projetos representam etapas importantes da minha evolução como desenvolvedor.
